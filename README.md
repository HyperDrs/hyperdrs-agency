# HYPERDRS - Digital Marketing Agency for Running Shoe Stores

A high-performance, mobile-first, blue-themed digital marketing agency website for HYPERDRS, catering specifically to the running shoe store niche.

## Features
- **Mobile-First Responsive Grid**: Built using CSS Flexbox and Grid designed for phones first, scaling up smoothly to desktop viewports.
- **Speed Optimized**: Pure HTML5, CSS3, and ES6+ Vanilla JavaScript with zero third-party framework overhead, ensuring fast loading speeds.
- **Niche Focus Content**: Tailored features for running shoe shops (Local Retailer SEO, sneaker releases, run club marketing, and Klaviyo loyalty automations).
- **Audit Request Lead Capture**: Custom interactive contact form optimized to capture details from retail shop owners (locations, budget spend, etc.).
- **Theme Accents**: Highly professional primary blue styling structure representing stability and speed.

## Project Structure
- `index.html` - Semantic markup structure.
- `style.css` - Custom variables, responsive grid settings, and animations.
- `app.js` - Mobile drawer toggle, category filtering, and mock form submit.
- `assets/` - Generated high-fidelity images mapping local shoe retail boutiques.
